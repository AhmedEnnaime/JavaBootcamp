# JavaBootcamp

- Variables

- Loops

- Arithmetic operators

- Arrays

- Strings

- Enums

- JDBC

- IOStreams

- Generics

- Annotations

- Multi-threading

- Concurency

- Serialization

- Synchronization

- Networking

Still working to improve and discover more advanced features in Java
